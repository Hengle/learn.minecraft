## 安装gradle

* 安装 bin的

https://gradle.org/releases/

* 配置环境变量

```
GRADLE_HOME 
D:\bin\gradle-6.3
```

```
GRADLE_USER_HOME
D:\bin\gradle-6.3\_userhome
```

```
PATH
%GRADLE_HOME%\bin
```


## 下载 mdk

* 搜索 Minecraft forge
* https://files.minecraftforge.net/
* 下载mdk


## 下载 Intelijj IDE 

* 选择安装社区版本

https://www.jetbrains.com/idea/
